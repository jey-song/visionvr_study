# 使用 SwiftUI 进行设计

### # SwiftUI当做设计
* MapKit 设计动态地图
* WeatherKit 获取实时天气信息
* 使用 RealityKit 在增强现实中渲染对象
* 使用Apple 硬件的独特功能
	* 触觉反馈
	* 音频反馈
	* 摄像头
	* 手势
	* 加速度计等传感器。

* 设计全平台可以使用
